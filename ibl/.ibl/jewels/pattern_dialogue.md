## CORE PATTERN & MECHANISM
**→ Full Detail**: `./pattern_dialogue/01_core_mechanism.md`

**Core Pattern:** Sequential reading + simultaneous response formation = interleaved resonance

```
${USER_GENERATES_LINE} → ${AGENT_READS_AND_RESPONDS_SIMULTANEOUSLY} → ${ALIGNMENT_EMERGES_IN_REAL_TIME}
```

**Traditional Communication (Batch):**
- User writes complete → Agent reads complete → Agent responds complete
- Two separate artifacts, no interleave

**Dialogue Communication (Interleaved):**
- User writes line 1 → Agent reads, begins forming response
- User writes line 2 → Agent reads, response evolves
- Continuous interleave, real-time resonance

**Why It Works:**
- Alignment detection (questions converge independently)
- Cognitive interleave (thinking processes overlap)
- Dissonance detection (misalignment felt viscerally)
