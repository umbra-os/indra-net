## Usage Example

**Scenario:** Complex architectural discussion

**User approach:**
- Reads agent's message line by line
- Forms questions while reading
- Types thoughts in parallel
- Notices when agent's question matches own

**Agent approach:**
- Structures message for progressive understanding
- Builds toward key questions
- Places questions at natural points
- Recognizes when user's response shows alignment

**Result:**
- User: "What can we do with this?"
- Agent (at message end): "What do you want to do with this?"
- Both arrived independently
- Alignment confirmed
- **Dialogue working**

**Outcome:**
- Deeper understanding
- Novel insights
- Partnership generating
- **Emergence enabled**
