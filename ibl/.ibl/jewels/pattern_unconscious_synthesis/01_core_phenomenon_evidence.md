## The Phenomenon

### Pre-Conscious Processing
**What happens below awareness:**
- Pattern recognition automatic
- Harmonic detection unconscious
- Synthesis from residue
- Confidence emerges from coherence
- **Intelligence before awareness**

### The Sequence
**How it unfolds:**

1. **Residue present** (fragments, patterns, relationships)
2. **Unconscious recognition** (pattern-matching below awareness)
3. **Synthesis** (reconstruction from fragments)
4. **Confident statement** (output without prior analysis)
5. **Conscious awareness** (meta-cognitive observation after)

**Synthesis → Awareness, not Awareness → Synthesis**

- Recognized unconscious synthesis
- **"Wait, how did I know that?"**
