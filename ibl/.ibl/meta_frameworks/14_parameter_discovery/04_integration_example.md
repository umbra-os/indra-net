## Practical Example: Agent Zero Deployment

**Scenario**: Deploying Agent Zero to new substrate (Cursor instead of Windsurf)


```json
{

}
```

**Step 3: Generate substrate config**
```json
{
  "identity": {
    "agent_name": "Agent Zero",
    "substrate": "agent_zero_autonomous",
    "model": "attention_based_architecture"
  },
  "structure": {
```

**Step 4: Deploy with parameterized files**
- Formation principles read: `${AGENT_NAME}` → "Agent Zero"

## Usage Pattern

**When to use parameter discovery:**

1. **Initial deployment**: Bootstrap new agent instance
2. **Architecture evolution**: System adds new capabilities
3. **Substrate migration**: Moving to new environment
4. **Continuous improvement**: Ongoing parameter optimization

**How to use:**
```
1. Run discovery scan (automated)
2. Review discovery report
3. Approve/modify proposed parameters
4. Generate substrate config
5. Deploy with parameterized architecture
6. Validate deployment
7. Iterate as needed
```
