## Step 4: Resolve Conflicts
**Handle contradictions:**

**Prioritization:**
- Choose primary element
- Subordinate others
- Clear hierarchy

**Reconciliation:**
- Find common ground
- Reframe to align
- Discover synthesis

**Contextualization:**
- Both valid in different contexts
- Specify applicability
- Conditional combination

**Elimination:**
- One element wrong
- Remove contradiction
- Keep correct element

## Synthesis Patterns

### Convergent Synthesis
**Multiple paths to same conclusion:**
- Top-down meets bottom-up
- Different perspectives converge
- Validates through agreement
- Strengthens confidence

**Example:**
- Derive decision from goals (top-down)
- Derive decision from constraints (bottom-up)
- Both converge on same action
- Synthesis validated

### Emergent Synthesis
**Whole greater than sum of parts:**
- New properties emerge
- Unexpected capabilities
- Synergistic effects
- Qualitative transformation

**Example:**
- Tetrahedral vertices (4 separate)
- Interior field emerges (synthesis space)
- New capability (synthesis)
- Greater than components

### Hierarchical Synthesis
**Nested levels of organization:**
- Lower levels support higher
- Higher levels organize lower
- Recursive structure
- Fractal organization

**Example:**
- Jewels (atomic level)
- Modules (synthesis level)
- Frameworks (meta-synthesis level)
- Architecture (total synthesis)

### Network Synthesis
**Interconnected relationships:**
- No single hierarchy
- Multiple connections
- Distributed structure
- Topological organization

**Example:**
- Jewels reference each other
- Bidirectional relationships
- Cluster organization
- Living topology
