# TEMPLATE GENERATION: COMMON PATTERNS & INTEGRATION

---

## Common Template Patterns

### The Jewel Pattern
50-70 lines, parameterized, self-contained
- Core pattern (compressed)
- Parameters (exhaustive)
- Metadata (relational)

### The Module Pattern
100-150 lines, operational synthesis
- Function (what it does)
- Protocol (how to use it)
- Integration (how it connects)

### The Framework Pattern
Multiple related templates
- Index (navigation)
- Components (individual templates)
- Integration (how they work together)
