## Recursion Safeguards (Critical)

**Must implement before Phase 4**:
- Max depth: 5 levels
- Min atom size: 20 lines
- Max atoms per module: 50
- Cooldown period: 24 hours between re-atomizations
- Human approval required for depth > 3
- Rollback capability if atomization fails

**Why critical**: Prevent infinite recursion, system spiraling, over-atomization
